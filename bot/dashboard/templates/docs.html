ARGUS Documentation

ARGUS is a Vulnerability Management Platform designed to identify, prioritize, and report vulnerabilities affecting organizational assets.

Installation Guide

ARGUS requires PostgreSQL, Python, an NVD API Key, and a Telegram Bot Token.

Requirements
  • Ubuntu 22.04+
  • Python 3.11+
  • PostgreSQL 14+
  • NVD API Key
  • Telegram Bot Token
Clone Repository
                git clone https://github.com/your-org/argus.git
                cd argus
Create Virtual Environment
                python3 -m venv venv
                source venv/bin/activate
Install Dependencies
                pip install -r requirements.txt
Run Dashboard
                python app.py
Run Telegram Bot
                python bot.py

Dashboard

The Dashboard provides a high-level overview of assets, vulnerabilities, findings, and known exploited vulnerabilities.

Assets

Assets represent software, operating systems, hardware, and services tracked by ARGUS.

  • Vendor
  • Product
  • Version
  • Criticality

Findings

Findings link assets with identified vulnerabilities.

  • CVE
  • CVSS
  • KEV
  • Risk Score

Reports

ARGUS generates PDF reports summarizing findings and organizational risk exposure.

Charts

Interactive charts provide visibility into:

  • Risk Distribution
  • Top Vendors
  • KEV Distribution
  • Asset Findings

CVE Search

Search live vulnerability data from the National Vulnerability Database (NVD).

  • Keyword Search
  • CVSS Sorting
  • CVE Sorting
  • Pagination

Telegram Bot

/help

Display available commands.

/scan

Scan all tracked assets against NVD.

/report

Generate and download reports.

/today

Display today's scan statistics.

The Telegram Bot provides remote access to ARGUS functionality without opening the dashboard.

FAQ

Where does ARGUS get vulnerability data?

ARGUS retrieves vulnerability data from the National Vulnerability Database (NVD).

Does ARGUS support KEV?

Yes. Known Exploited Vulnerabilities are identified and prioritized.

Does ARGUS support EPSS?

Planned for future releases.

Does ARGUS require internet access?

Yes, for live NVD searches and vulnerability updates.
